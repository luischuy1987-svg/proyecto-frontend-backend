# Proyecto Frontend y Backend con Base de Datos

## Descripción
Este proyecto consiste en el desarrollo de una aplicación web que permite insertar información en una base de datos utilizando un frontend y un backend.

## Tecnologías utilizadas
- HTML
- JavaScript
- Node.js
- Express
- MariaDB
- GitHub

## Estructura del proyecto

proyecto-frontend-backend

backend
- server.js
- db.js

frontend
- index.html

README.md

## Funcionamiento

1. El usuario ingresa sus datos en un formulario en la página web (frontend).
2. El formulario envía la información al servidor (backend).
3. El backend recibe los datos y los procesa.
4. Los datos se guardan en la base de datos.

## Conclusión

Este proyecto demuestra la conexión entre el frontend, el backend y la base de datos para insertar información en un sistema web.
